Customer Retention & Churn Analysis
Future Interns – Data Science & Analytics Task 2 (2026)**

📊 Project Overview
Analyzed 1,000 customer records from a SaaS subscription platform (2023–2024) to identify churn patterns, retention drivers, and actionable strategies to reduce customer loss.

🔑 Key Insights
- Free plan customers churn at 50.7% — highest risk segment with $0 LTV
- Pricing & budget issues account for 37.6% of all churn reasons
- Enterprise customers generate $4,451 avg LTV at only 5.7% churn rate
- Retained customers stay 2.4× longer than churned customers (440 vs 184 days)

🛠️ Tools Used
- Python (data generation & analysis)
- Chart.js (interactive visualizations)
- HTML/CSS (dashboard)
